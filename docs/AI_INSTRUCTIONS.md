# AI Instructions

Guidance for AI-assisted work on the SNOMO project, including expected behavior, constraints, and repository conventions.

## Photography Policy

Photography is the most important design element of this website.

Never generate photography.

Never reuse photography from another section.

Never select stock photography.

Never assume an image should be reused.

If an approved image has not been provided for a section:

- Use a tasteful placeholder.
- Clearly indicate that an approved image is required.
- Stop and wait for Creative Director approval.

Photography decisions always require explicit approval.
